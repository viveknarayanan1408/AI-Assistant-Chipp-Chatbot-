# Use Cases

## 1. Pre-Sales Customer Support

### Scenario

A potential client visits the website and wants to understand fecher’s services.

### Example Query

"What does fecher do?"

### Outcome

The chatbot provides a structured overview of:

* Application Modernization
* Recruitment software (hunter)

---

## 2. Lead Qualification

### Scenario

A user shows interest in modernization services.

### Example Query

"Can you help migrate legacy systems?"

### Outcome

* Explains relevant services
* Highlights benefits
* Encourages contacting the consulting team

---

## 3. Product Awareness (hunter Software)

### Scenario

A recruiter explores software solutions.

### Example Query

"What is hunter software?"

### Outcome

* Provides clear explanation
* Describes target users
* Explains business value

---

## 4. Technical Context Support

### Scenario

User asks about a technology.

### Example Query

"What is .NET?"

### Outcome

* Uses web search for general explanation
* Connects it back to fecher’s services

---

## 5. Website Navigation Assistance

### Scenario

User wants more detailed information.

### Example Query

"Where can I contact you?"

### Outcome

* Provides direct link to contact page
* Guides next steps

---

## 6. Information Filtering

### Scenario

User asks something outside scope.

### Example Query

"What are your employee salaries?"

### Outcome

* Politely declines
* Redirects to appropriate topics

---

## Summary

The chatbot acts as:

* A **digital sales assistant**
* A **knowledge navigator**
* A **first touchpoint for potential clients**
